# Autism
This project implements a complete in silico drug discovery workflow for Autism Spectrum Disorder (ASD), focusing on five synaptic proteins with strong genetic links to ASD: mTOR, SHANK3, NLGN3, NRXN1, and SYNGAP1.
The pipeline performs:
Target selection with ChEMBL bioactivity data
ADMET filtering for CNS-penetrant drug-like compounds
Molecular docking with AutoDock Vina
Results visualization and validation
